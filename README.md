 Tastenkombinationen & Hinweise


Dieses Programm enthält keine Minecraft-Texturen und keine Texturen aus Resourcepacks.  
Aus Copyright gründen.

Texturen müssen selber einmalig geladen werden.
Prinzipiell könnt ihr alle Resourcenpacks und sogar eigene Textursets nutzen und diese dann mit der Blockzuordnung 
einer Block ID zuweisen. Habe es bisher allerdings nur mit 16x16 Texturen probiert.



Bedienung:
Linksklick => Block/Textur setzen (ein Feld)
Linksklick halten + Ziehen=>  Malen über mehrere Felder (Paint-Drag)
Rechtsklick =>  Entfernen / Radieren (je nach Tool-Modus)

Ebenentools:
Alt + Mausrad =>aktive Ebene wechseln  (im Arbeitsbereich)
Mittlere Maustaste => Neue Ebene erstellen
Alt + Mittlere Maustaste im Arbeitsbereich=> Aktuelle Ebene löschen
Strg + Linksklick auf eine Ebene => Ebene sichtbar/unsichtbar schalten  


Vorschau:
Ctrl + Mausrad =>Zoom
