
   
Dieses Programm enthält aus Copyrightgründen keine Minecraft-Texturen und keine Texturen aus Resourcepacks. 
Texturen müssen selber einmalig geladen werden.

Prinzipiell könnt ihr alle Resourcenpacks, sowie eigene Textur-sets, nutzen und diese dann mit der Blockzuordnung 
einer Block ID zuweisen.
